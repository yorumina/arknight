# Game Introduction

本專題目標為復刻《明日方舟》核心玩法，聚焦於：

- 網格地圖與關卡編排
- 高地位／地面位部署限制
- 敵方路徑、速度、停頓與方向控制
- 可擴充的模組化架構（方便後續串接 UI、戰鬥與資料檔）

目前已完成一版可執行的關卡設計核心程式（Python），可用於快速驗證地圖與敵人移動設計。

[遊戲連結](https://www.youtube.com/watch?v=PFY0K0MLzwU)
