# Game Introduction

本專題目標為復刻《明日方舟》核心玩法，聚焦於：

- 網格地圖與關卡編排
- 高地位／地面位部署限制
- 敵方路徑、速度、停頓與方向控制
- 可擴充的模組化架構（方便後續串接 UI、戰鬥與資料檔）

目前已完成一版可遊玩的 C++ / PTSD 遊戲原型，包含關卡載入、敵人波次、幹員部署、攻擊/技能、勝敗流程、HUD 操作與關卡設計工具。

目前主要未完成項目為音效/特效素材整合、獨立道具系統、專案測試案例、展示影片與最終簡報。

[遊戲連結](https://www.youtube.com/watch?v=PFY0K0MLzwU)
